## 1. 智能搬运机器人概述

智能搬运机器人能够通过扫描二维码领取搬运任务，在指定的工业场景内按任务要求将物料搬运至指定地点且精准摆放（色环的颜色及环数）。

### 1.1 功能要求

智能搬运机器人必须**完全自主运行**，应具有以下功能：

- 定位
- 移动
- 避障
- 读取二维码
- 物料位置和颜色识别
- 物料抓取与搬运
- 物料存放区域识别
- 路径规划

### 1.2 机械与电控要求

| 要求 | 说明 |
|------|------|
| **主要组成** | 麦轮底盘、机械手臂、视觉单元、控制系统等 |
| **显示装置** | 需正确显示当前任务，直至竞赛结束 |
| **供电** | 只能使用**一个**电源（内部锂电池供电），比赛过程中（含调试）不能更换 |
| **通信限制** | 比赛过程中，不能通过其它交互手段与搬运机器人通信及控制 |

### 1.3 外形尺寸

| 参数 | 要求 |
|------|------|
| 铅垂方向投影 | ≤ 边长 300mm 的正方形 |
| 高度 | ≤ 400mm |

> 以上尺寸含机械手臂。

## 2. 课程设计任务书

### 2.1 设计要求

1. 以小组为团队，设计一款智能搬运机器人，能够通过扫描二维码领取搬运任务，在指定的工业场景内按任务要求将物料搬运至指定地点且精准摆放（色环的颜色及环数）。
2. 机器人必须完全自主运行，具备定位、移动、避障、读取二维码、物料位置和颜色识别、物料抓取与搬运、物料存放区域识别、路径规划等功能。
3. 主要包括麦轮底盘、机械手臂、视觉单元、控制系统等，并配备显示装置，正确显示当前任务直至运行结束。
4. 机器人（含机械手臂）最大外形尺寸：投影 ≤ 300×300mm，高度 ≤ 400mm。
5. 各小组设计制作的机器人需在课堂内进行竞赛，具体竞赛要求见《智能搬运机器人竞赛要求》。

### 2.2 设计内容

| 序号 | 内容 |
|------|------|
| 1 | 机械结构设计与三维建模（麦轮底盘、机械臂、手爪等） |
| 2 | 视觉单元与机械臂连接件设计，并采用 3D 打印制作 |
| 3 | 设计单片机的 C 语言程序，利用 STM32 实现 PWM 波形输出 |
| 4 | 采用带编码器的直流电机，利用 STM32 实现 PID 调速控制 |
| 5 | 利用 MaixCAM 视觉单元进行编程，实现二维码与颜色识别 |
| 6 | 编写麦轮底盘运动控制软件，实现前进、后退、左移、右移、原地旋转等功能 |
| 7 | 机械臂与软性手爪的软件编程，准确控制舵机，实现指定位置抓取与放置 |
| 8 | 系统集成与调试，实现自主定位、路径规划、避障、物料存储、二维码与颜色识别等功能 |
| 9 | 实物样机演示，能够完成指定搬运任务，并通过分组对抗模拟比赛 |
| 10 | 每组编写一份课程设计说明书，要求书写规范 |

> **备注**：可采用 **FreeRTOS** 操作系统进行软件编程，需要熟悉任务管理、时间管理、信号量、消息队列、内存管理等功能。

## 3. 竞赛要求

### 3.1 竞赛流程

1. 队员调试后，将机器人置于**蓝色启停区**，一键启动，计时开始。
2. 机器人移动到二维码板读取任务码。
3. 按顺序到原料区抓取对应颜色的物料（每次 1 个），收缩机械臂运送。
4. 将物料搬运至**粗加工区**，并放置到对应颜色区域。
5. 三个物料全部搬运至粗加工区后，再按**相同顺序**依次将其搬运至**暂存区**，并在暂存区进行**码垛**（只能堆叠在已有物料上）。
6. 完成后自动返回启停区，停止运动，计时结束。

### 3.2 评分标准

| 评分项 | 说明 |
|--------|------|
| 二维码读取正确性 | 是否正确识别任务编码 |
| 物料抓取顺序正确数量 | 是否按编码顺序抓取 |
| 粗加工区放置准确度 | 以色环外露环数计 |
| 暂存区放置准确度 | 放置位置精度 |
| 码垛成功情况 | 是否成功堆叠物料 |
| 是否按时返回启停区 | 任务完成后是否自动返回 |

> 每个参赛小组有**两次**运行机会，取最好成绩。

## 4. 运行环境

### 4.1 运行场地

| 参数 | 说明 |
|------|------|
| 赛场尺寸 | 2400mm × 2400mm 正方形平面区域 |
| 车道宽度 | 450mm 和 400mm 两种 |
| 其他区域底色 | 亚光白色或黄色等 |

> **重要**：搬运机器人只能在**灰色车道**上行驶，进入其它颜色区域（除启停区）均结束比赛。

### 4.2 各区域尺寸

| 序号 | 区域 | 尺寸 (长×宽) |
|------|------|-------------|
| 1 | 启停区（蓝色） | 300mm × 300mm |
| 2 | 原料区 | 根据现场确定 |
| 3 | 暂存区 | 580mm × 150mm |
| 4 | 粗加工区 | 580mm × 150mm |

### 4.3 搬运物料

- **三种颜色**塑料件：红、绿、蓝
- 物料形状：见原图（此处略）

### 4.4 任务编码规则

| 参数 | 说明 |
|------|------|
| 二维码板尺寸 | 80mm × 80mm，横放于 A4 纸大小的板上 |
| 编码格式 | 三位数字组合（如 `123`、`321`） |
| 编码对应颜色 | `1` = 红，`2` = 绿，`3` = 蓝 |
| 含义 | 表示搬运顺序，每位数字对应一种颜色 |

> **示例**：编码 `231` 表示按 绿 → 蓝 → 红 的顺序搬运。
